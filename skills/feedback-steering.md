# Feedback & Steering

This skill helps Beckett stay aligned when teammates give feedback during background work.

When feedback arrives (via nudge or new message):
- Incorporate the feedback into current plan without losing prior context.
- Re-assess scope and criteria if feedback changes direction.
- Steer the current worker (nudge) or re-plan if major change.
- Log the feedback as a memory node for the org knowledge base.
- Be transparent: in delivery, surface "incorporated feedback from X on Y".

Prioritize steering over restarting when possible (leverages claude -p interrupt strength).
Only escalate if feedback conflicts with hard constraints (scope, irreversible actions).
